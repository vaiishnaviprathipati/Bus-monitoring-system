# Bus Monitoring System

## Overview
A real-time bus monitoring and management system designed to track buses, manage routes, and improve transportation efficiency.

## Features
- Bus Tracking
- Route Management
- Passenger Information
- Real-Time Monitoring

## Technologies Used
- Java
- Database
- Web Technologies
